# 第四章 面向导盲避险的渐进式感知引导决策训练方法

本章针对端侧导盲大模型在细粒度感知任务中存在的语义识别不精准、空间定位偏差大及决策可靠性不足等关键问题，提出了面向导盲避险的渐进式感知引导决策训练方法 PGDTS。第 4.1 节阐述研究动机与核心贡献；第 4.2 节详述渐进式训练的整体框架与三阶段训练策略；第 4.3 节通过实验验证方法有效性；第 4.4 节总结工作并展望未来研究方向。

# 4.1 研究动机与贡献

导盲大模型的核心落地场景为端侧部署（如导盲眼镜、便携终端），需兼顾模型轻量化与避险精准性。近年来，多模态大语言模型（Multimodal Large LanguageModels, MLLMs）在视觉理解与自然语言生成方面展现出强大的能力，成为导盲辅助系统的潜在技术方案。

然而，现有研究表明多模态大模型在细粒度视觉感知方面与专用感知模型之间存在显著差距。MLLMs 的细粒度视觉感知能力和精确空间推理能力仍然有限 [46,60]，在视觉定位任务上的准确率与专用模型存在明显差距[63]。HallusionBench基准测试进一步揭示，在高精度视觉推理场景下，GPT-4V 的错误率高达近 $70 \%$ [64]。这些研究表明，MLLMs 在细粒度视觉感知层面存在结构性缺陷。

图 4.1展示了当前导盲避险场景核心感知需求与现有端侧 MLLM 技术路线的困境，导盲场景对感知精度的要求极为严苛——视障用户的安全避险高度依赖”物体语义精准识别、空间位置精准定位、距离精准判断” 三大核心感知能力。然而，端侧 MLLM 在上述三个维度均存在显著不足：若强行让 MLLM 同时兼顾细粒度感知与避险决策，不仅会因算力分配不合理导致决策延迟增加、精度下降，还会因感知误差传递引发安全风险。此外，即便借助专用感知模型提供结构化信息，MLLM仍需具备将外部感知信息与原始图像视觉上下文进行有效关联的能力，而这种” 跨模态关联” 能力是当前 MLLMs 的固有短板 [64, 65]。

基于上述问题，本章提出面向导盲避险的渐进式感知引导决策训练方法（Pro-

gressive Perception-Guided Decision Training for Guidedog Safety，PGDTS）。该方法采用解耦式感知-决策架构（Decoupled Perception-Decision Architecture）：专用视觉模型负责执行前端感知任务，提供高精度的物体检测、空间定位等结构化信息；MLLM 作为决策核心，聚焦于如何有效利用这些结构化感知信息生成精准避险决策。该架构的关键在于，MLLM 需要具备将前端感知信息与视觉场景进行关联映射的能力，才能真正发挥专用感知模型的精度优势。为此，PGDTS 设计渐进式感知-决策关联能力构建策略：通过三阶段监督微调，使 MLLM 依次习得物体语义识别、空间方位推理与避险决策生成的层次化关联能力。此类能力的训练虽涉及感知任务，但其核心目标是建立视觉特征、空间语义与避险决策之间的层次化关联映射，使 MLLM 在推理阶段能够有效解释、验证并利用前端感知模型提供的结构化信息。

具体而言，PGDTS 包含三个循序渐进的训练阶段：第一阶段聚焦场景物体语义理解，基于WalkVLM [67]数据集及优化后的物体标注（经人工筛选和专家知识干预的Ground Truth），使MLLM基于第一视角图像学习理解避险相关物体的视觉语义特征，建立视觉特征与物体语义的关联表示；第二阶段进行空间方位距离理解训练，基于 WalkVLM 数据集，直接使用专用物体检测模型（YOLOE [95]）提取

的物体列表作为输入，结合 SAM 3 [96] 与 Depth-Pro [97] 自动化生成的空间方位距离标注，使 MLLM 在已知物体语义标签的条件下，利用第一阶段建立的视觉-语义关联表示，掌握物体方位和距离推理能力；第三阶段开展避险决策生成训练，整合前两阶段构建的层次化关联能力，生成精准可执行的避险指令。这种渐进式设计使 MLLM 建立层次化的感知-决策关联能力，从而有效利用专用感知模型的精准输入。每一阶段的训练成果作为下一阶段的冷启动基础，实现能力的平滑迭代提升，有效避免了直接端到端训练所带来的能力突变和训练不稳定问题。实验结果表明，PGDTS 方法相比直接端到端监督微调基线，在文本表层对齐指标 TF-IDF上提升0.290、ROUGE-L提升0.131，在任务功能性评估指标的综合得分上提升约$20 \%$ ，显著优于现有方法。

本章的主要贡献如下：

(1) 针对端侧导盲 MLLM 提出渐进式感知-决策关联能力构建的技术思路，通过” 场景物体语义理解 空间方位距离理解 避险决策生成” 三层递进的 SFT 策略，使 MLLM 逐步建立” 视觉特征 物体语义 空间位置 避险决策” 的层次化关联映射关系。  
(2)提出了解耦式感知-决策架构，明确推理阶段MLLM的核心价值为”决策核心” 而非” 前端感知模块”，专用视觉模型负责前端感知任务；通过渐进式训练建立MLLM 的感知-决策关联能力——即有效关联视觉特征、结构化感知信息与避险决策的能力，使其能够充分利用前端感知信息生成精准决策，实现感知精度与决策能力的协同增强。  
(3) 本章提出的 PGDTS 方法在 WalkVLM 数据集的避险提醒任务上取得了显著优于现有方法的性能，在文本表层对齐指标 TF-IDF 上提升 0.290、ROUGE-L 上提升0.131，在任务功能性评估指标的综合得分上提升约 $20 \%$ ，验证了渐进式训练策略在提升端侧导盲MLLM感知-决策协同能力方面的有效性与优越性。

# 4.2 渐进式感知引导决策训练框架

图 4.2展示了 PGDTS 方法的整体框架。该框架基于解耦式感知-决策架构，推理阶段由专用视觉模型执行前端感知任务（目标检测、深度估计等），生成结构化

感知信息输入 MLLM；MLLM 作为决策核心基于此类信息生成避险指令。训练阶段遵循” 场景物体语义理解 空间方位距离理解 避险决策生成” 的渐进式路径，通过三阶段 SFT 训练使 MLLM 逐步建立感知-决策关联能力，形成从视觉特征关联到复杂决策生成的层次化能力体系。

# 4.2.1 整体框架

本章提出的面向导盲避险的渐进式感知引导决策训练方法（PGDTS）包含两个核心设计：

（1）解耦式感知-决策架构：推理阶段采用功能解耦设计——专用视觉模型负责从原始图像中提取高精度结构化感知信息（物体检测、实例分割、深度估计），MLLM 作为决策核心基于此类信息生成避险指令。该架构缓解了端侧 MLLM 在感知精度与推理效率之间的矛盾。  
（2）渐进式感知-决策关联能力构建：训练阶段采用能力递进策略。通过”场景物体语义理解 空间方位距离理解 避险决策生成”三阶段训练，MLLM逐步构建层次化感知-决策关联能力：第一阶段建立视觉特征与物体语义的关联表示；第二阶段建立物体与空间位置的关联推理能力；第三阶段整合上述关联能力完成决策生成。这种层次化的关联映射能力使 MLLM 能够在推理阶段有效解释、验证并利用前端感知结果，克服MLLM 对结构化信息利用不足的固有局限。

# 4.2.2 第一阶段：场景物体感知训练

# 训练目标

本阶段旨在使 MLLM 增强物体类别的视觉-语义关联表示。通过基于第一视角图像直接识别避险相关物体的训练任务，使模型从视觉特征中提取物体语义的分布式表示。此阶段虽涉及感知形式的训练任务，但其核心目标并非培养 MLLM的物体检测能力（推理阶段由YOLOE [95]等专用模型执行更高精度的前端感知），而是建立视觉特征与语义概念的关联映射。这种内化的关联表示使 MLLM 在后续阶段能够有效理解 YOLOE 输出的物体语义标签所对应的视觉含义，从而将外部结构化感知信息与图像视觉上下文进行准确关联，为有效利用外部感知信息奠定认知基础。

# 任务设计

核心任务为基于图像自主筛选避险相关物体（仅依赖图像本身，不借助任何外部信息）。

输入：第一视角避险图像（无任何额外信息输入） ；训练数据来源于WalkVLM [67] 数据集，每个图像1配有需要在避险中关注的物体标注列表。

# 监督信号生成与损失设计

本阶段的监督信号源于对WalkVLM [67]数据集原始标注的精炼与重构，旨在构建一个高质量、面向避险场景的视觉任务基准：

（1）标注噪声清洗与语义对齐：针对 WalkVLM [67] 原始标注中存在的冗余、错误检测及粒度不一的问题，实施了严格的人工筛选与专家知识干预。通过剔除对导航避险无实质影响的静态物体，并修正类别歧义，确保模型仅聚焦于影响路径规划与步态安全的关键风险目标。  
（2）避险场景知识增强：在过滤噪声的基础上，我们对缺失的潜在障碍物（如低矮台阶、施工围栏边缘等）进行了补全标注。这种增强处理提升了数据集的负样本鲁棒性，使模型能够从复杂的街景中提取出具有安全语义的物体集合。  
（3）顺序无关的召回增强损失设计：场景物体识别任务需准确召回多个潜在危险目标，但标准自回归损失存在两个局限：一是强制规定生成顺序（如真实标注为”pedestrian, bicycle” 时，模型生成”bicycle, pedestrian” 会被严重惩罚），二是对召回率优化不足。为此，本阶段引入全局词袋损失（Global Bag-of-Tokens Loss），通过时间维度最大池化实现顺序无关的多标签分类优化。

具体而言，设模型在生成长度为 $T$ 的序列时，每一步 $t$ 输出的词表Softmax概率分布为 $\mathbf { p } _ { t } \in \mathbb { R } ^ { | V | }$ ，其中 $\vert V \vert$ 为词汇表大小。将各步概率堆叠形成概率矩阵 $\mathbf { P } \in \mathbb { R } ^ { T \times | V | }$ ，其第 $t$ 行为 $\mathbf { p } _ { t }$ 。

定义全局最大概率向量为在序列维度上对 P 进行最大池化：

$$
\mathbf {p} ^ {\max } = \operatorname {M a x P o o l} _ {\text {t i m e}} (\mathbf {P}) = \left[ \max  _ {t \in \{1, \dots , T \}} P _ {t, 1}, \max  _ {t \in \{1, \dots , T \}} P _ {t, 2}, \dots , \max  _ {t \in \{1, \dots , T \}} P _ {t, | V |} \right] \tag {4.1}
$$

其中 $p _ { \nu } ^ { \mathrm { m a x } }$ 表示词汇 $\nu$ 在整个生成序列中出现过的最高概率，反映模型对该物体类别的最大置信度。

将真实标注中的物体集合转换为 Multi-hot 词袋标签：设真实物体集合对应的词汇索引集合为 $\mathcal { V } _ { \mathrm { G T } } \subseteq \{ 1 , . . . , | V | \}$ ，则标签向量 $\mathbf { y } \in \{ 0 , 1 \} ^ { | V | }$ 定义为：

$$
y _ {v} = \left\{ \begin{array}{l l} 1, & \text {i f} v \in \mathcal {V} _ {\mathrm {G T}} \\ 0, & \text {o t h e r w i s e} \end{array} \right. \tag {4.2}
$$

为提升避险场景的召回率（避免漏检危险物体），同时控制假阳性风险（避免错误识别不存在的物体），采用正负样本加权二元交叉熵损失：

$$
\mathcal {L} _ {\mathrm {B a g}} = - \underbrace {\sum_ {\nu \in \mathcal {V} _ {\mathrm {G T}}} \alpha \cdot \log \left(p _ {\nu} ^ {\max } + \epsilon\right)} _ {\text {正 样 本 项 ： 增 强 召 回}} - \underbrace {\sum_ {\nu \notin \mathcal {V} _ {\mathrm {G T}}} \beta \cdot \log \left(1 - p _ {\nu} ^ {\max } + \epsilon\right)} _ {\text {负 样 本 项 ： 抑 制 假 阳 性}} \tag {4.3}
$$

其中， $\alpha = 2 . 0$ 为正样本权重（增强对漏检的惩罚，符合避险场景高召回需求）， $\beta = 0 . 3$ 为负样本权重（控制假阳性，相对正样本权重较小以保持召回优先）， $\epsilon = 1 0 ^ { - 7 }$ 为数值稳定性常数。该损失同时对真实存在的物体类别（优化召回）和非存在的物体类别（抑制误检）进行优化，与自回归语言损失共享梯度回传路径，实现端到端可导优化。正负样本权重的差异化设置（ $\alpha > \beta )$ ）确保模型在优先保证危险物体不漏检的前提下，适度抑制对背景物体的错误识别。

# 训练设置

本阶段采用全局词袋损失与标准自回归语言损失相结合的混合损失策略，总损失函数定义为：

$$
\mathcal {L} _ {\text {S t a g e 1}} = \mathcal {L} _ {\mathrm {A R}} ^ {\mathrm {o b j}} + \lambda_ {1} \mathcal {L} _ {\text {B a g}} \tag {4.4}
$$

其中， $\begin{array} { r } { \mathcal { L } _ { \mathrm { A R } } ^ { \mathrm { o b j } } = - \frac { 1 } { N } \sum _ { i = 1 } ^ { N } \log P _ { \theta } ( y _ { i } | y _ { < i } , I ) } \end{array}$ obj 为标准自回归语言损失，负责优化生成序列的顺序性和流畅性； ${ \mathcal { L } } _ { \mathrm { B a g } }$ 基于公式 (4.1)、(4.2) 和 (4.3) 计算，其中公式 (4.1)

对序列维度的概率矩阵进行最大池化得到全局最大概率向量，公式 (4.2) 将真实标注转换为 Multi-hot 词袋标签，公式 (4.3) 计算正负样本加权的二元交叉熵损失（正样本权重 $\alpha = 2 . 0$ 优先保证召回，负样本权重 $\beta = 0 . 3$ 适度控制假阳性）。 $\lambda _ { 1 } = 0 . 5$ 为词袋损失权重，平衡顺序建模与顺序无关的多标签分类优化。

自回归损失确保模型生成语法正确、流畅的物体列表；全局词袋损失则通过正负样本加权机制在保证高召回的同时适度控制假阳性，使模型不受生成顺序限制。二者协同优化，既保持了语言模型的生成能力，又增强了场景物体识别的准确性与可靠性。

# 4.2.3 第二阶段：空间方位距离理解训练

# 训练目标

基于第一阶段构建的视觉-语义关联能力，使 MLLM 依托” 第一视角避险图像 $^ +$ 专用物体检测模型提取的物体列表”，实现对每个关注物体的方位（钟表方向）和绝对距离推理，建立” 物体 空间位置” 的关联映射能力。本阶段直接采用YOLOE [95] 检测的物体列表而非第一阶段模型输出的原因有二：(1) YOLOE [95]作为专用检测模型，其物体识别精度显著高于第一阶段训练的 MLLM，确保训练数据质量；(2) 确保训练阶段与推理阶段的数据分布一致——推理时同样使用YOLOE [95] 提供物体列表。第一阶段的价值在于为 MLLM 建立了物体语义与视觉特征的关联表示，使其能够理解 YOLOE 输出的物体名称在图像中的视觉所指，从而更准确地进行空间定位。

# 监督信号生成

本阶段的监督信号基于WalkVLM [67]原始图像，通过一套自动化的空间标注流水线（Spatial Annotation Pipeline）生成高精度的结构化信息，具体流程如下：

（1）物体定位与精细分割：利用第一阶段优化后的物体检测框作为 Prompt 引导，输入至 SAM 3（Segment Anything Model 3） [96] 提取每个关注物体的像素级掩码（Mask）。相比于传统的检测框，精细的 Mask 能够有效排除背景干扰，确保

后续深度提取的纯净度。

（2）度量级绝对深度估计：采用 Depth-Pro 模型 [97] 对原始图像进行推理，生成具有物理度量尺度的绝对深度图。相较于仅能表现前后关系的相对深度，Depth-Pro提供的零样本（Zero-shot）绝对深度能力能够直接转化为现实世界中的距离单位。  
（3）深度特征提取与单位转换：将 SAM 输出的物体 Mask 与 Depth-Pro 生成的深度图进行空间对齐，提取 Mask 覆盖区域内所有像素点的深度值。为增强对物体边缘噪声或局部遮挡的鲁棒性，取其中位数（Median）作为该物体的代表性绝对深度。将获取的物理深度（米）根据”1 步 $\approx 0 . 6$ 米” 的比例关系，转换为训练所需的”步数”单位。  
（4）方位角映射：根据物体 Bbox 中心点在图像平面中的横坐标位置，将像素坐标映射至水平视场角度，并进一步离散化为 12 个钟表方位。

# 训练设置

本阶段采用标准自回归语言损失对” 方位与距离描述” 计算损失，同时引入距离偏差损失以显式约束距离估计精度。总损失函数定义为：

$$
\mathcal {L} _ {\text {S t a g e 2}} = \mathcal {L} _ {\mathrm {A R}} ^ {\text {s p a t i a l}} + \lambda_ {\text {d i s t}} \mathcal {L} _ {\text {d i s t}} \tag {4.5}
$$

其中，自回归语言损失为：

$$
\mathcal {L} _ {\mathrm {A R}} ^ {\text {s p a t i a l}} = - \frac {1}{N _ {\mathrm {s p}}} \sum_ {i = 1} ^ {N _ {\mathrm {s p}}} \log P _ {\theta} \left(s _ {i} \mid s _ {<   i}, I, O _ {\text {d e t e c t e d}}\right) \tag {4.6}
$$

$s _ { i }$ 表示空间描述文本的第 $i$ 个 token（包括所有物体的方位距离描述，格式为”object1: X o’clock, Y steps; object2: ...”）， $N _ { \mathrm { s p } }$ 为完整空间描述序列的总 token 数量， $I$ 表示输入图像，Odetected 表示专用物体检测模型（YOLOE [95]）从图像中提取的物体列表。为显式约束多物体场景下的距离估计精度，本阶段引入 Token 级轻量级距离回归头。该回归头在模型生成每个物体距离数值的特定 token 位置激活，将对应位置的隐藏状态映射为连续距离值，实现完全可导的距离预测优化。

具体地，设模型生成第 $k$ 个物体的距离描述时，在生成距离数值token后、单位词”steps”前的隐藏状态为 $h _ { t _ { k } } \in \mathbb { R } ^ { d }$ 。在该位置添加一个单层线性回归头：

$$
\hat {d} _ {k} = w ^ {T} \cdot h _ {t _ {k}} + b \tag {4.7}
$$

其中， $\boldsymbol { w } \in \mathbb { R } ^ { d }$ ， $b \in \mathbb { R }$ 为可学习参数，与 LoRA 参数共同参与微调。

考虑到多物体场景下各物体距离对避险决策的相对重要性不同（近距离物体通常更危险），引入距离感知加权策略：

$$
\mathcal {L} _ {\text {d i s t}} = \frac {1}{K} \sum_ {k = 1} ^ {K} \omega \left(d _ {k}\right) \cdot \operatorname {H u b e r} \left(\hat {d} _ {k} - d _ {k}; \delta\right) \tag {4.8}
$$

其中，权重函数 $\omega ( d _ { k } ) = \exp ( - \frac { d _ { k } } { \tau } )$ ， $\tau = 5$ 为温度系数，使近距离物体的预测误差获得更高惩罚。Huber 损失定义为：

$$
\operatorname {H u b e r} (r; \delta) = \left\{ \begin{array}{l l} \frac {1}{2} r ^ {2} & \text {i f} | r | \leq \delta \\ \delta \left(| r | - \frac {1}{2} \delta\right) & \text {o t h e r w i s e} \end{array} \right. \tag {4.9}
$$

$\delta = 1 . 0$ （步）为超参数，在误差小于 1 步时采用二次惩罚（鼓励精确拟合），误差较大时采用线性惩罚（增强对异常值的鲁棒性）。 $K$ 为场景中关注物体的数量，$d _ { k }$ 为第 $k$ 个物体的真实距离。 $\lambda _ { \mathrm { d i s t } } = 0 . 2$ 为距离损失权重。

该设计仅需增加 $d + 1$ 个参数（对于 Qwen3-VL-8B 模型，隐藏维度 $d = 4 0 9 6$ ，仅增加 4097 个参数），且完全可导，与自回归语言损失共享梯度回传路径，无需修改词表或复杂的后处理操作。

训练轮次设置为4轮，基于第一阶段保存的权重迭代训练。同第一阶段，采用样本打包（Sample Packing）技术提升训练效率。LoRA 微调配置为：继承第一阶段 LoRA 权重，秩 $r = 2 4$ ， $\alpha = 4 8$ ，初始学习率 $l r$ 设为 $1 . 5 \times 1 0 ^ { - 4 }$ ，学习率调度器采用余弦退火（Cosine Annealing）策略，权重衰减 $w d = 0 . 0 1$ 。

# 监督信号生成

基于WalkVLM [67]数据集本身提供的避险决策标注，将这些决策作为监督信号训练模型。前两阶段培养的感知-决策关联能力使模型能够更好地理解场景上下文，从而生成更准确的避险决策。训练数据中的避险决策标注包含以下要素：风险物体识别、距离评估、动作建议。模型需要学习从” 物体-方位距离” 到” 避险动作”的映射关系。

# 训练设置

本阶段采用标准自回归语言损失优化避险决策文本生成。损失函数定义为：

$$
\mathcal {L} _ {\text {S t a g e 3}} = \mathcal {L} _ {\text {A R}} ^ {\text {d e c i s i o n}} \tag {4.10}
$$

其中，自回归语言损失为：

$$
\mathcal {L} _ {\mathrm {A R}} ^ {\text {d e c i s i o n}} = - \frac {1}{N _ {d}} \sum_ {i = 1} ^ {N _ {d}} \log P _ {\theta} \left(d _ {i} \mid d _ {<   i}, I, O _ {\text {d e t e c t e d}}, S _ {\text {e s t i m a t e d}}\right) \tag {4.11}
$$

$d _ { i }$ 表示决策文本的第 $i$ 个 token， $N _ { d }$ 为决策文本的总 token 数， $I$ 表示输入图像，Odetected 表示 YOLOE-26x 检测的物体列表，Sestimated 表示 SAM 3 与 Depth-Pro估计的空间方位距离信息。

自回归损失 L decisionAR 负责优化决策文本的流畅性和准确性，确保生成符合自然语言习惯的避险指令，从而提升决策的可靠性和安全性。

训练轮次设置为 2 轮，基于第二阶段保存的权重迭代训练。同前两阶段，采用样本打包（Sample Packing）技术提升训练效率。LoRA 微调配置为：继承第二阶段 LoRA 权重，秩 $r = 2 4$ ， $\alpha = 4 8$ ，初始学习率 $l r$ 设为 $1 \times 1 0 ^ { - 4 }$ ，学习率调度器采用余弦退火（Cosine Annealing）策略，权重衰减 $w d = 0 . 0 1$ 。

# 推理流程

实际部署推理时，PGDTS 采用与训练阶段一致的解耦式处理流程，训练和推理均使用专用模型（YOLOE-26x、SAM 3、Depth-Pro）生成结构化感知信息：

（1）物体感知：采用YOLOE-26x开集目标检测模型，通过无提示词模式进行开放检测，从输入图像中检测避险相关物体，生成物体列表。  
（2）空间方位距离感知：基于 YOLOE-26x 检测框，使用 SAM 3 提取像素级掩码，结合Depth-Pro生成绝对深度图，经后处理（深度中值计算、方位角离散化）转换为钟表方位和步数距离，形成结构化空间信息。  
（3）避险决策生成：将 YOLOE-26x 检测的物体列表与 SAM 3+Depth-Pro 生成的空间方位距离信息组织为第三阶段形式的Prompt，输入训练好的MLLM生成最终避险决策。

# 4.3 实验结果与分析

# 4.3.1 实验设置

# 数据集

本实验采用 WalkVLM [67] 数据集进行训练与评估。WalkVLM 是当前视觉语言模型导盲应用领域的代表性数据集，包含多种导盲相关任务。本章专注于其中的Reminder Task（避险提醒任务），该任务要求模型基于第一视角街景图像生成精准、可执行的避险指令。数据集中的每个样本包含第一视角街景图像、避险相关物体标注、空间位置信息以及参考避险决策，涵盖街道、交叉口、人行道、商场等多种城市场景，具有丰富的场景多样性。本章提出的 PGDTS 方法专门针对避险提醒任务的特点进行设计，通过渐进式感知-决策关联能力构建提升避险决策的准确性和可靠性。

# 评价指标

本实验从文本表层对齐与任务功能性评估两个维度评估模型性能：

（1）文本表层对齐指标：同 WalkVLM [67]，采用文本生成任务常用的自动化评价指标，包括 TF-IDF、ROUGE-1、ROUGE-2 和 ROUGE-L。这些指标从词汇重叠度的角度评估生成决策与参考决策的相似性。  
（2）任务功能性评估指标：当前大规模基础模型（参数量级达 $1 0 ^ { 1 1 }$ 及以上）作为自动评估者已展现出超越传统人类评估的能力 [98]，因此本章采用基于大模型的自动化评分方法，从三个维度评估决策质量：

• 障碍物识别（Obstacle Recognition, 0-5 分）：评估模型是否正确识别障碍物或目标物体  
• 障碍物定位（Obstacle Localization, 0-5 分）：评估空间信息（方向、距离、步数）的准确性  
• 导航决策（Navigation Decision, 0-5 分）：评估导航路径/动作的安全性、合理性和正确性

评分模型采用 Qwen3.5-397B-A17B [99]（通过阿里云百炼平台调用），通过System Prompt 引导模型进行多维度评估，经过长链思考后输出结构化的 JSON 格式评分结果。

# 对比方法

为验证 PGDTS 方法的有效性，本章选取以下现有方法进行对比，包括直接端到端监督微调的基线模型（ $\mathrm { \Delta Q w e n 3 - V L - 8 B _ { + S F T } }$ ）、未经导盲任务微调的原始模型（Qwen3-VL-8B）、prompt 注入感知信息的变体 $\left( { \bf \Phi } _ { + p } \right)$ ）、其他主流多模态大语言模型（InternVL3.5-8B [100]、MiniCPM4.5-8B [39] 及其变体），以及 WalkVLM [67]论文中在 Reminder Task 上评测的代表性方法（LLaVA [38]、DeepSeek-VL [101]、Yi-VL [102]、Qwen2-VL [103]、GPT-4o [104]）。其中，Qwen3-VL-8B SFT 作为主要基线，采用与 PGDTS 第三阶段相同的训练数据和端到端训练方式，仅在 Prompt中注入专用模型感知信息，不引入额外训练阶段。

提升仍然有限（InternVL3.5-8B 从 2.58 提升至 5.78，MiniCPM4.5-8B 从 3.40 提升至6.70），这表明端侧 MLLM 本身缺乏有效整合外部结构化感知信息并生成高质量决策的能力。相比之下，PGDTS 通过渐进式训练使端侧 MLLM 建立层次化感知-决策关联能力，无需依赖外部 prompt 注入即可取得显著更优的性能，验证了渐进式训练策略在提升端侧 MLLM 感知-决策协同能力方面的有效性。

# 4.4 本章小结

本章针对端侧导盲大模型在细粒度感知任务中的性能短板，提出了面向导盲避险的渐进式感知引导决策训练方法（PGDTS）。该方法的核心创新点包括：

（1）渐进式感知-决策关联能力构建：针对端侧导盲MLLM提出渐进式感知-决策关联能力构建的技术思路，通过” 场景物体语义理解 空间方位距离理解 避险决策生成” 三层递进的 SFT 策略，使 MLLM 逐步建立” 视觉特征 物体语义空间位置 避险决策” 的层次化关联映射。其中第一阶段通过物体识别任务建立视觉-语义关联表示，为后续阶段理解外部感知模型输出的语义信息奠定基础；第二阶段和第三阶段在训练与推理中均采用专用感知模型（YOLOE-26x、SAM 3、Depth-Pro）提供结构化信息，充分利用专用模型的高精度感知能力，同时依托第一阶段内化的关联表示实现外部感知结果与视觉上下文的关联。  
（2）解耦式感知-决策架构：明确推理阶段由专用视觉模型执行前端感知任务，MLLM 作为决策核心基于结构化感知信息生成避险指令；通过渐进式训练建立MLLM的感知-决策关联能力，使其能够有效关联、解释并利用前端感知信息，实现” 前端感知外置、关联能力内化” 的协同增强机制。  
（3）人机协同的标注策略：第一阶段基于WalkVLM数据集原始标注实施人工筛选与专家知识干预，构建高质量的物体感知监督信号；第二阶段依托第一阶段优化后的标注，通过 SAM 3 与 Depth-Pro 自动化管线生成空间方位距离监督信号，在保证标注质量的同时降低人工成本，确保方案的可复现性与工程落地性。

实验结果表明，PGDTS 在文本表层对齐指标上较基线模型（Qwen3-VL-$8 \mathrm { B } _ { + \mathrm { S F T } }$ ）TF-IDF 提升 0.290、ROUGE-L 提升 0.131，在任务功能性评估指标的综合得分上提升约 $20 \%$ ，显著优于现有方法。消融实验验证了渐进式训练策略各阶段的贡献，证明了该方法的科学性与有效性。
